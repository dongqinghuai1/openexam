## 3. 索引设计

### 3.1 主键索引
所有表都有主键索引(id)，自动创建。

### 3.2 常用查询索引

```sql
-- 用户查询
CREATE INDEX idx_user_phone ON tb_user(phone);
CREATE INDEX idx_user_status ON tb_user(status);

-- 学生查询
CREATE INDEX idx_student_phone ON tb_student(phone);
CREATE INDEX idx_student_status ON tb_student(status);
CREATE INDEX idx_student_grade ON tb_student(grade);

-- 教师查询
CREATE INDEX idx_teacher_phone ON tb_teacher(phone);
CREATE INDEX idx_teacher_status ON tb_teacher(status);

-- 排课查询
CREATE INDEX idx_schedule_class_date ON tb_schedule(edu_class_id, date);
CREATE INDEX idx_schedule_teacher_date ON tb_schedule(teacher_id, date);
CREATE INDEX idx_schedule_status ON tb_schedule(status);

-- 课时查询
CREATE INDEX idx_hours_account_student ON tb_student_hours_account(student_id);
CREATE INDEX idx_hours_flow_account ON tb_hours_flow(account_id);

-- 订单查询
CREATE INDEX idx_order_student ON tb_order(student_id);
CREATE INDEX idx_order_status ON tb_order(status);
CREATE INDEX idx_order_no ON tb_order(order_no);
```

## 4. 外键约束

```sql
-- 示例外键
ALTER TABLE tb_student ADD CONSTRAINT fk_student_user 
  FOREIGN KEY (creator_id) REFERENCES tb_user(id);

ALTER TABLE tb_schedule ADD CONSTRAINT fk_schedule_class 
  FOREIGN KEY (edu_class_id) REFERENCES tb_class(id);

ALTER TABLE tb_class ADD CONSTRAINT fk_class_course 
  FOREIGN KEY (course_id) REFERENCES tb_course(id);

ALTER TABLE tb_class ADD CONSTRAINT fk_class_teacher 
  FOREIGN KEY (teacher_id) REFERENCES tb_teacher(id);
```

## 5. 视图设计

### 5.1 学生课时汇总视图

```sql
CREATE VIEW v_student_hours_summary AS
SELECT 
    s.id as student_id,
    s.name as student_name,
    s.phone,
    c.id as course_id,
    c.name as course_name,
    ha.total_hours,
    ha.used_hours,
    ha.total_hours - ha.used_hours as remaining_hours,
    ha.expire_date
FROM tb_student_hours_account ha
JOIN tb_student s ON ha.student_id = s.id
JOIN tb_course c ON ha.course_id = c.id;
```

### 5.2 班级课表视图

```sql
CREATE VIEW v_class_schedule AS
SELECT 
    cl.id as class_id,
    cl.name as class_name,
    co.name as course_name,
    t.name as teacher_name,
    sc.date,
    sc.start_time,
    sc.end_time,
    sc.room,
    sc.status
FROM tb_schedule sc
JOIN tb_class cl ON sc.edu_class_id = cl.id
JOIN tb_course co ON sc.course_id = co.id
JOIN tb_teacher t ON sc.teacher_id = t.id;
```

## 7. 数据迁移

使用 Django migrations 管理数据库版本。

```bash
# 生成迁移
python manage.py makemigrations

# 执行迁移
python manage.py migrate

# 创建超级用户
python manage.py createsuperuser
```
