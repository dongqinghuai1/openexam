# 项目概览

## 1. 项目信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 在线教学教务管理平台 |
| 项目代号 | EduAdmin |
| 版本 | V1.0 |
| 开发阶段 | MVP核心闭环 |
| 启动日期 | 2026-04-07 |

## 2. 项目背景

随着在线教育行业的快速发展，需要一套完整的教务管理系统来支撑日常教学运营。本系统旨在打通教学、教务、测评、经营四大业务板块，实现从招生报名到课后服务的完整业务闭环。

## 3. 技术架构

### 3.1 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端 | Python/Django | 4.2+ |
| 后端API | Django REST Framework | 3.14+ |
| 前端管理后台 | Vue3 + Element Plus | 3.3+ / 2.4+ |
| 前端课堂端 | Taro (React) | 3.6+ |
| 数据库 | PostgreSQL | 14+ |
| 缓存 | Redis | 6+ |
| RTC | 腾讯会议 | WebRTC + SDK |
| 消息队列 | Celery | 5.3+ |

## 4. 模块划分

### 4.1 业务模块

| 模块 | 说明 | 优先级 |
|------|------|--------|
| 用户权限 | 登录/注册/角色/权限 | P0 |
| 学生管理 | 学生档案/状态管理 | P0 |
| 教师管理 | 教师档案/擅长科目 | P0 |
| 课程管理 | 课程/章节/课时 | P0 |
| 班级管理 | 班级/分班/绑定 | P0 |
| 排课管理 | 日历排课/冲突检测/调课 | P0 |
| 在线课堂 | 腾讯会议接入 | P0 |
| 录屏回放 | 自动录制/回放 | P0 |
| 课时管理 | 账户/流水/扣减 | P0 |
| 订单管理 | 创建/支付/状态 | P0 |
| 通知 | 短信/微信模板 | P1 |

### 4.2 模块依赖

```

课程管理 ─┬─► 班级管理
```
