# 在线教学教务管理平台 (EduAdmin)

## 项目概述

- **项目名称**: 在线教学教务管理平台
- **版本**: V1.0 (MVP核心闭环)
- **创建日期**: 2026-04-07
- **状态**: 开发中
- **Git仓库**: https://github.com/dongqinghuai1/openexam

---

## 项目目标

实现"能招生建档、能排课、能上课、能录屏、能扣课时、能收款"的核心业务闭环。

---

## 技术架构

| 层级 | 技术选型 |
|------|----------|
| 前端管理后台 | Vue3 + Element Plus (端口: 3000) |
| 课堂端(多端) | Taro (React) - 教师端/学生端/家长端 |
| 后端 | Python/Django 4.2 + DRF 3.14 (端口: 8000) |
| 数据库 | PostgreSQL 14+ |
| RTC | 腾讯会议 SDK (Mock开发中) |
| 缓存 | Redis 6+ |
