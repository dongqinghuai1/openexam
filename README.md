# 在线教学教务管理平台（EduAdmin）

在线教学教务管理平台是一个围绕教培场景构建的多角色系统，目标是覆盖招生建档、排课、上课、录屏、扣课时、收款等核心业务闭环。

## 技术栈

### 后端

- Python 3.8+
- Django 4.2+
- Django REST Framework 3.14+
- PostgreSQL 14+
- Redis 6+

### 前端

- 管理后台：Vue3 + Element Plus + Vite
- 课堂端：Taro（React），覆盖教师端、学生端、家长端

### 第三方服务

- 腾讯会议 SDK（当前以 Mock 为主）
- 短信服务
- 支付网关
