# 部署指南

## 目标环境

- Python 3.8+
- Node.js 18+
- PostgreSQL 14+
- Redis 6+
- Docker / Docker Compose（推荐）

## 本地开发启动

### 后端

```bash
cd 04_后端开发/03_业务代码
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver 8000
```

### 管理后台

```bash
cd 03_前端开发/01_管理后台
npm install
npm run dev
```

### 课堂端

```bash
cd 03_前端开发/02_教师端
npm install
npm run dev:h5

cd 03_前端开发/03_学生端
npm install
npm run dev:h5

cd 03_前端开发/04_家长端
npm install
npm run dev:h5
```

## Docker 启动

```bash
docker-compose up -d
```

默认暴露端口：

- 前端：`3000`
- 后端：`8000`
- PostgreSQL：`5432`
- Redis：`6379`

## 当前注意事项

1. Docker 编排以 PostgreSQL + Redis 为目标环境。
2. Django 当前默认开发配置仍保留 SQLite 回退逻辑，部署前需确认环境变量已正确注入。
3. 腾讯会议相关能力仍以 Mock 为主，生产前需要补齐真实配置。
