# 快速开始

## 环境要求

- Python 3.8+
- Node.js 18+
- PostgreSQL 14+（推荐）
- Redis 6+

## 后端启动

```bash
cd 04_后端开发/03_业务代码
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver 8000
```

## 管理后台启动

```bash
cd 03_前端开发/01_管理后台
npm install
npm run dev
```

访问：`http://localhost:3000`

## 课堂端启动

```bash
cd 03_前端开发/02_教师端
npm install
npm run dev:h5

cd 03_前端开发/03_学生端
npm install
npm run dev:h5

cd 03_前端开发/04_家长端
npm install
npm run dev:h5
```

## Docker 启动

```bash
docker-compose up -d
```
