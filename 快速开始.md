# 快速开始指南

## 环境要求

- Python 3.8+
- Node.js 18+
- PostgreSQL 14+

## 后端启动

```bash
# 1. 进入后端目录
cd 04_后端开发/03_业务代码

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置数据库
# 编辑 settings.py 中的 DATABASES 配置

# 5. 执行迁移
python manage.py migrate

# 6. 创建超级用户
python manage.py createsuperuser

# 7. 启动服务
python manage.py runserver 8000
```

## 前端启动

```bash
# 管理后台 (Vue3)
cd 03_前端开发/01_管理后台
npm install
npm run dev  # 访问 http://localhost:3000

# 教师端 (Taro)
cd 03_前端开发/02_教师端
npm install
npm run dev:h5  # 访问 http://localhost:10086

# 学生端 (Taro)
cd 03_前端开发/03_学生端
npm install
npm run dev:h5

# 家长端 (Taro)
cd 03_前端开发/04_家长端
npm install
npm run dev:h5
```

## 初始化数据

```bash
# 创建测试数据
python manage.py shell < init_data.py
```
