# 管理员操作手册

## 角色: 超级管理员/校长

---

## 1. 系统概述

本系统为在线教学教务管理平台，超级管理员拥有最高权限，可管理所有模块。本手册介绍超级管理员的核心操作功能。

## 2. 登录与权限

### 2.1 登录系统

1. 访问管理后台地址
2. 输入用户名和密码
3. 如开启验证码，输入验证码
4. 点击登录

### 2.2 权限说明

超级管理员拥有以下权限：
- 用户管理：创建、编辑、删除用户
- 角色权限：配置角色和权限
- 学生管理：全部学生数据
- 教师管理：全部教师数据
- 课程管理：全部课程
- 班级管理：全部班级
- 排课管理：全部排课
- 在线课堂：全部课堂
- 录屏回放：全部录像
- 课时管理：全部课时
- 订单管理：全部订单
- 退款管理：审批退款
- 统计分析：全部数据
- 系统配置：系统设置
- 日志审计：查看操作日志

## 3. 核心功能操作

### 3.1 用户管理

#### 创建用户

1. 进入「系统管理」→「用户管理」
2. 点击「新增用户」按钮
3. 填写用户信息：
   - 用户名（必填）
   - 真实姓名（必填）
   - 手机号（必填）
   - 邮箱
   - 角色（必选）
   - 部门
4. 点击「保存」

#### 分配角色

1. 在用户列表中找到目标用户
2. 点击「编辑」按钮
3. 在「角色」下拉框中选择角色
4. 点击「保存」

### 3.2 角色权限管理

#### 创建角色

1. 进入「系统管理」→「角色管理」
2. 点击「新增角色」按钮
3. 填写角色信息：
   - 角色名称（必填）
   - 角色编码（必填，唯一）
   - 描述
4. 勾选权限菜单
5. 点击「保存」

#### 配置数据权限

在角色编辑页面，可以配置数据权限范围：
- 全部数据：可查看所有数据
- 部门数据：仅可查看本部门数据
- 仅自己：仅可查看自己创建的数据

### 3.3 学生管理

#### 创建学生档案

1. 进入「教务管理」→「学生管理」
2. 点击「新增学生」按钮
3. 填写学生信息：
   - 姓名（必填）
   - 性别（必填）
   - 出生日期
   - 年级（必填）
   - 学校
   - 家长姓名（必填）
   - 家长手机（必填）
4. 点击「保存」

#### 查看学生详情

1. 在学生列表中点击学生姓名
2. 可查看：基本信息、课时余额、订单记录、上课记录

### 3.4 教师管理

#### 创建教师档案

1. 进入「教务管理」→「教师管理」
2. 点击「新增教师」按钮
3. 填写教师信息：
   - 姓名（必填）
   - 性别（必填）
   - 手机号（必填）
   - 学历
   - 专业
   - 擅长科目（多选）
   - 入职日期
4. 点击「保存」

### 3.5 课程管理

#### 创建课程

1. 进入「教务管理」→「课程管理」
2. 点击「新增课程」按钮
3. 填写课程信息：
   - 课程名称（必填）
   - 课程编码
   - 所属科目（必选）
   - 总课时（必填）
   - 单次时长（分钟）
   - 价格
   - 课程描述
4. 点击「保存」

#### 管理章节

1. 在课程详情页点击「章节管理」
2. 添加大章节
3. 在大章节下添加小结

### 3.6 班级管理

#### 创建班级

1. 进入「教务管理」→「班级管理」
2. 点击「新增班级」按钮
3. 填写班级信息：
   - 班级名称（必填）
   - 班级编码
   - 选择课程（必选）
   - 授课教师（必选）
   - 最大人数
   - 开班日期
4. 点击「保存」

#### 分配学生

1. 进入班级详情
2. 点击「添加学生」
3. 选择学生并确认

### 3.7 排课管理

#### 创建排课

1. 进入「教务管理」→「排课管理」
2. 点击「新增排课」或选择日期
3. 选择班级、教师、时间段、教室
4. 系统会自动检测冲突
5. 点击「保存」

#### 调课操作

1. 在排课列表中找到课次
2. 点击「调课」按钮
3. 选择新时间或新教师
4. 填写调课原因
5. 提交审批

### 3.8 订单管理

#### 查看订单

1. 进入「财务管理」→「订单管理」
2. 可查看所有订单
3. 支持按状态、日期、学生筛选

#### 审批退款

1. 进入「财务管理」→「退款管理」
2. 找到退款申请
3. 点击「审批」
4. 选择通过或拒绝
5. 如通过，资金将原路返回

### 3.9 统计分析

#### 查看数据看板

1. 进入「统计分析」→「数据看板」
2. 可查看核心指标：
   - 招生数据
   - 收入数据
   - 上课数据
   - 课时消耗

#### 查看财务报表

1. 进入「统计分析」→「财务报表」
2. 查看收入汇总
3. 查看欠费统计
4. 导出报表

### 3.10 系统配置

#### 基本设置

1. 进入「系统管理」→「系统配置」
2. 配置项：
   - 系统名称
   - Logo
   - 时区
   - 验证码开关

#### 课时规则

配置课时扣减规则：
- 正常上课扣课时数
- 请假规则
- 迟到处理
- 旷课处理

#### 通知模板

配置短信/微信通知模板

### 3.11 日志审计

#### 查看操作日志

1. 进入「系统管理」→「日志审计」
2. 可查看：
   - 登录日志
   - 操作日志
3. 支持按时间、用户、操作类型筛选

## 4. 常见问题

### 4.1 用户无法登录

1. 检查账号密码是否正确
2. 检查用户状态是否为「启用」
3. 检查是否分配了角色

### 4.2 排课冲突

系统会自动检测教师、教室、班级的时间冲突。如遇冲突，需要调整时间或更换教师/教室。

### 4.3 退款处理

退款需要审批通过后原路返回，退款后该订单关联的课时会被冻结或扣减。
