# Todo List 汇总

## 说明

`todo_list/` 作为项目执行清单的唯一任务索引，保留各角色的当前状态、阻塞项和下一步工作。

为避免信息失真，本目录不再维护重复的周计划模板、日报模板和夸张的百分比进度。项目总览请看根目录 `README.md`，运行方式请看 `快速开始.md` 与 `部署指南.md`。

## 使用规则

1. 每个角色文档只维护四个区块：已完成、进行中、待处理、阻塞项。
2. 已经完成的事项写结果，不再重复保留未来周模板。
3. 汇总状态必须和正文一致，避免出现“上方已完成、下方待开始”的冲突。
4. 需要复盘或过程记录时，单独写到对应业务目录，不回填到 TodoList 模板中。
